# Contributing to PixelPro

Thanks for contributing! This project is built for learning and collaboration.

## Quick Start

1. Fork the repo
2. Clone: `git clone https://github.com/YOUR_USERNAME/pixelpro.git`
3. Install: `npm install` (frontend) and `cd server && npm install` (backend)
4. Start servers: `cd server && npm run dev` + `npm run dev`
5. Make changes, commit, push, create PR

## Setup

1. Fork and clone the repo
2. Install dependencies: `npm install` and `cd server && npm install`
3. Copy `server/.env.example` to `server/.env`
4. Start servers: `cd server && npm run dev` + `npm run dev`

## Workflow

1. Create branch: `git checkout -b feature/your-feature`
2. Make changes and test locally
3. Commit: `git commit -m "Add: your feature"`
4. Push and create PR

## Code Style

- Write clear, commented code
- Follow existing patterns
- Keep functions small
- Use TypeScript types
